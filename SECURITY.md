# Security policy

## Supported versions

Document which versions currently receive security updates.

| Version | Supported |
| --- | --- |
| main | yes |

## Reporting a vulnerability

Do not report vulnerabilities in public issues.

Send reports to <security-contact@example.com> and include:

- affected version or commit
- reproduction steps
- impact
- any suggested mitigation

We will acknowledge receipt and follow up as soon as possible.
